# Machine Learning Syllabus

## Prerequisites
- **Python**: Data structures, NumPy, Pandas, Matplotlib, Seaborn
- **Math**: Basic statistics, probability, and linear algebra (recommended)
- **Programming Skills**: Familiarity with loops, functions, and object-oriented programming concepts

## LEVEL–1: Noob's Guide to Machine Learning

| Topic | Description | Resources |
|-------|-------------|-----------|
| **Day 1: Introduction to Machine Learning** | Definition and types (supervised, unsupervised, reinforcement). How to formulate ML problems, key terminologies, classification overview, Linear Regression as classifier, extending to multi-class, basics of ANN (forward pass, gradient descent, loss, backpropagation). | |
| **Day 2: End-to-End ML Pipeline** | Data preprocessing pipeline: handling missing values, encoding categorical data, normalization, feature engineering, and selection. Building a preprocessing pipeline using the Heart Disease dataset. Training a simple MLP model. | |
| **Day 3: Model Accuracy and Evaluation** | Understanding hyperparameters and tuning (Grid Search, Random Search). Metrics: Accuracy, Precision, Recall, F1-score, ROC-AUC. Confusion matrix analysis and visualization. | |

### ML Project–001 (7 days)
**Project Topic: Heart Disease Prediction**

**Guidelines:**
- Perform data preprocessing and feature engineering
- Build a neural network classifier using scikit-learn
- Evaluate results with multiple metrics
- Visualize performance and interpret model outcomes

## LEVEL–2: Core Machine Learning

| Topic | Description | Resources |
|-------|-------------|-----------|
| **Day 1: Supervised Learning Algorithms** | KNN, Decision Trees, Random Forests, ensemble methods, and bias–variance tradeoff. Regularization (L1, L2). Model comparison and selection. | |
| **Day 2: Model Optimization and Feature Engineering** | Advanced feature engineering, cross-validation (K-Fold, Stratified), feature importance analysis, hyperparameter tuning, and model generalization concepts. | |
| **Day 3: Unsupervised Learning and Dimensionality Reduction** | K-Means and Hierarchical clustering, PCA, and t-SNE for visualization. Evaluation of clustering performance and use cases in feature compression. | |

### ML Project–002 (7 days)
**Project Topic: Customer Segmentation**

**Guidelines:**
- Use a retail or e-commerce dataset
- Preprocess, normalize, and apply clustering (K-Means)
- Visualize clusters and analyze results using PCA

## LEVEL–3: Advanced Machine Learning Foundations (Deep Learning Core)

| Topic | Description | Resources |
|-------|-------------|-----------|
| **Day 1: Transition from ML to DL** | Understanding the limitations of classical ML. Deep Learning motivation, architecture overview, layers, activations, loss functions, and optimizers (SGD, Adam). | |
| **Day 2: Introduction to PyTorch** | PyTorch basics: tensors, autograd, defining neural networks, training loop, loss computation, and optimizer setup. Implementing a simple ANN. | |
| **Day 3: MNIST Neural Network Training** | Full training pipeline for MNIST digits. DataLoader usage, backpropagation, loss optimization, regularization (dropout), performance visualization, and saving models. | |

### ML Project–003 (7 days)
**Project Topic: MNIST Handwritten Digit Classifier**

**Guidelines:**
- Implement a neural network from scratch using PyTorch
- Train and validate on MNIST dataset
- Evaluate accuracy and visualize predictions
- Document findings and learning curve

## Learning Outcomes

By completing all three levels, participants will be able to:
- Understand and implement key ML and DL algorithms
- Build complete data preprocessing and model training pipelines
- Apply evaluation metrics effectively
- Implement and train neural networks using PyTorch
- Create portfolio-ready ML projects and transition smoothly into specialized tracks (NLP, CV)
